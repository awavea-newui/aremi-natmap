Data Workbench

When a data set is added to the map via the Data Catalogue, a legend for that data set will appear at the top of your Data Workbench (pictured above). From the Data Workbench you can:

  • Set the order in which data is shown on the map. To do this, simply click on the title of a data set and drag it to a new position in your Workbench.
  • Toggle the visibility of added data sets. To do this, click on the checkbox opposite your prefered data set title.
  • Zoom to the geographical extent of an added data set
  • Set the opacity of individual data sets
  • Remove data sets from the map. Note: removed data sets can always be re-added via the Data Catalogue.

This tab allows you to search for a place or a dataset. When entering a search string, you will get first results related to ‘Locations’ and underneath, a link directing you to search the data catalogue. When clicking on Data Catalogue, a new window screen will be opened with only the data catalogue search results.

Place Name Search

Enter a name in the Search box to show a list of place names related to this search string. Those names shown under Locations have been found using the Bing Maps location search. Those names shown under Official Place Names have been found in the Geoscience Australia Gazetteer of official place names.

Click on any of these listed names to show that address, place or geographic feature in the map view. You can zoom to anywhere in the world in this way.
Data Set Name Search

When entering a search string in the Search Box, you will get a list of items related to ‘locations’ and underneath a link to a data catalogue specific search. A new window screen will be opened and results related to data catalogues search only will be displayed.

Click any of the check box for a Data Set to cause that Data Set to be shown on the Map View. Then click on the Data Set name itself to cause the Map View to zoom to a view which just encloses all of the features in that Data Set.
